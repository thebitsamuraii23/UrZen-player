## Как использовать

### Смена языка вручную
1. Нажмите на иконку **⚙️ Настройки**
2. Найдите опцию **Локализация** / **Language**
3. Выберите нужный язык из списка
4. Интерфейс изменится сразу же

### Автоматическая смена языка
При первом открытии приложение автоматически:
1. Определит язык вашего браузера
2. Установит соответствующий язык интерфейса
3. Сохранит выбор для будущих сеансов

## Технические детали

### Система локализации
- **Тип**: Простая система на основе объектов JavaScript
- **Селекторы**: HTML элементы с атрибутом `data-t="key"`
- **Динамическое применение**: Функция `applyLanguage()` работает без перезагрузки страницы

### Обработка ошибок
```javascript
// Если язык не найден:
if (!t) {
    state.lang = 'en';  // Fallback на английский
    return applyLanguage();  // Переприменяем
}
```

### Проверка поддерживаемых языков
```javascript
// detectBrowserLanguage() проверяет:
const supportedLanguages = ['en', 'ru', 'es', 'de', 'fr'];
```
