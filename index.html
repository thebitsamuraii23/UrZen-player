# UrZen

Минималистичный музыкальный плеер с реальным рабочим флоу: локальные треки, Navidrome, умные плейлисты и Zen Mode.

Vanilla TS Subsonic API IndexedDB JWT Auth
More on GitHub

## Почему это удобно

  • Одна очередь для локальной и серверной музыки.
  • Контекстные действия прямо в списке треков.
  • Мини-плеер и Zen Mode без лишних переходов.
  • Массовый Select для работы с плейлистами.

## Что внутри

  • Library: локальная медиатека и drag & drop.
  • Playlists: локальные и серверные коллекции.
  • Home: рекомендации, продолжение прослушивания, быстрый поиск.
  • Music tab: прямой доступ к Navidrome-каталогу.

## Техстек

Frontend: HTML + CSS + TypeScript
Backend: Node + Express
Storage: IndexedDB (Dexie)
API: Subsonic-compatible (Navidrome)
Auth: JWT + user scopes

## Быстрый flow

1. Добавляешь локальные треки или подключаешь Navidrome
2. Отмечаешь нужное через Select
3. Кидаешь пачкой в плейлист / избранное
4. Запускаешь Zen Mode и слушаешь без отвлечений
Смысл UrZen: меньше кликов, больше музыки.

## Новые функции

  • Полная миграция проекта на TypeScript: frontend, backend и модули.
  • AI Playlist Builder: создаёт плейлисты по описанию (жанры, артисты, настроение, количество треков).
  • Smart Shuffle: подбирает и ставит в очередь релевантные треки из вашей Navidrome-библиотеки.
  • Mini Queue с drag-and-drop: можно менять порядок треков прямо перетаскиванием.
  • Мобильный режим без старого fullscreen-плеера: упор на mini player + стабильный скролл.
  • Полностраничный mobile scroll: Home и плейлисты скроллятся как единый экран.
