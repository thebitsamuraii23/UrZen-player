## 📦 Установка и запуск

### Требования
```bash
Node.js 14+
npm или yarn
Браузер с поддержкой WebAudio API
```

### Быстрый старт
```bash
# Клонировать репозиторий
git clone <repo>
cd html-player

# Установить зависимости
npm install

# Запустить сервер
npm start

# Открыть в браузере
http://localhost:3001
```

### Docker (опционально)
```bash
# Собрать образ
docker build -t html-player .

# Запустить контейнер
docker run -p 3001:3001 html-player
```

## 🎯 Основные возможности

### 1. Поиск музыки
```
1. Нажмите на иконку поиска (🔍)
2. Введите название песни, артиста или альбома
3. Увидите результаты:
   - 📁 Из локальной библиотеки (загруженные файлы)
   - 🌐 Из Navidrome (streaming)
4. Кликните на результат
5. Музыка начинает играть
```

### 2. Загрузка музыки
```
1. Нажмите на "Import Tracks" 
2. Выберите MP3 файлы
3. Файлы загружаются в IndexedDB
4. Появляются в библиотеке
```

### 3. Воспроизведение
```
1. Выберите трек (поиск или библиотека)
2. Трек добавляется в очередь
3. Нажмите "Play"
4. Используйте кнопки управления:
   - ⏯ Play/Pause
   - ⏭ Next
   - ⏮ Previous
   - 🔄 Shuffle
   - 🔁 Repeat
```

### 4. Плейлисты
```
1. Создайте плейлист ("New Playlist")
2. Добавляйте треки в плейлист
3. Воспроизводите плейлист
4. Сохраняется в LocalStorage
```

## 🔧 API для разработчиков

### Поиск
```javascript
// Комбинированный поиск (local + navidrome)
const results = await window.performCombinedSearch("query");

// Только локальная библиотека
const local = await window.searchLocalLibrary("query");

// Только Navidrome
const navidrome = await window.searchNavidrome("query");
```

### Воспроизведение
```javascript
// Локальный трек
window.playTrack(trackId);

// Navidrome трек
await window.playNavidromeSong(songId, title, artist, album, coverUrl);
```

### Информация
```javascript
// Получить информацию о Navidrome песне
const details = await window.getSongDetails(songId);

// Получить URL потока
const streamUrl = window.getNavidromeStreamUrl(songId);

// Получить URL обложки
const coverUrl = window.buildNavidromeUrl('getCoverArt.view', {
  id: songId,
  size: 200
});
```

### Состояние плеера
```javascript
// Текущее состояние
console.log(window.state);

// Текущий трек
console.log(window.state.currentIndex);

// Библиотека
console.log(window.state.library);

// Плейлисты
console.log(window.state.playlists);
```
