UrZen

Modern Audio Player with Navidrome Integration

О плеере

UrZen - это современный минималистичный веб-плеер для управления вашей музыкальной библиотекой. Он поддерживает локальную загрузку треков, создание плейлистов, управление избранным и потоковую передачу музыки через Navidrome сервер.

Основные возможности

🎯 Управление плейлистами - создавайте и организуйте свою музыку
🔊 Bass Boost - контроль баса от 0 до 25 дБ
🔀 Перемешивание и повтор
❤️ Избранное - сохраняйте любимые треки
🌙 Zen Mode - погружение в музыку на полный экран
⌨️ Горячие клавиши для удобства

Источники музыки

• Локальная библиотека - загружайте свои треки
• Navidrome - личный музыкальный сервер
• Встроенный поиск - ищите во всех источниках одновременно
• IndexedDB - полная приватность ваших данных

Технологии

Backend: Node.js + Express
Frontend: Vanilla JavaScript (ES6+), HTML5, CSS3
Database: IndexedDB (локальное хранилище)
API: Subsonic API, JWT Authentication

Горячие клавиши

Используйте эти сочетания клавиш для удобного управления плеером

SPACE - Пауза/Воспроизведение
← → - Перемотка ±10 секунд
↑ ↓ - Громкость ±10%
F - Zen Mode
ESC - Выход из Zen / Закрыть справку
Ctrl → - Следующий трек
Ctrl ← - Предыдущий трек
