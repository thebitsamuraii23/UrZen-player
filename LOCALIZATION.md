# Локализация приложения Z BETA

## Поддерживаемые языки

Приложение Z BETA поддерживает следующие языки:

| Язык | Код | Флаг |
|------|-----|------|
| English | `en` | 🇺🇸 |
| Русский | `ru` | 🇷🇺 |
| Español | `es` | 🇪🇸 |
| Deutsch | `de` | 🇩🇪 |
| Français | `fr` | 🇫🇷 |

## Переключение языка

### Через настройки приложения
1. Откройте **Настройки** (Settings gear icon)
2. Найдите опцию **Локализация** (Localization)
3. Выберите нужный язык из выпадающего списка
4. Язык изменится автоматически

### Автоматическое определение
При первом открытии приложения язык устанавливается автоматически в зависимости от язык браузера:
- Браузер на английском → **English**
- Браузер на русском → **Русский**
- Браузер на испанском → **Español**
- Браузер на немецком → **Deutsch**
- Браузер на французском → **Français**
- Другие языки → **English** (по умолчанию)

## Хранение настроек

Выбранный язык сохраняется в:
1. **IndexedDB** (основное хранилище)
2. **localStorage** (резервное)

Это обеспечивает сохранение языка при повторном открытии приложения.

## Переводимые элементы

В приложении переводятся следующие элементы:
- Название вкладок (Библиотека, Избранное, Коллекции)
- Кнопки (Создать, Отмена, Подтвердить и т.д.)
- Текст уведомлений и сообщений об ошибках
- Подсказки (tooltip hints)
- Названия элементов интерфейса

## Система перевода

Приложение использует собственную систему локализации (I18N) на основе:
- **Файл перевода**: `js/i18n.js`
- **Система меток**: `data-t` атрибуты в HTML
- **Функция применения**: `applyLanguage()` в `settings.js`

## Добавление нового языка

Чтобы добавить новый язык:

1. Откройте `js/i18n.js`
2. Добавьте новую секцию с кодом языка:

```javascript
export const I18N = {
    // ... существующие языки ...
    xx: {  // где xx - код языка
        library: "Перевод",
        favorites: "Перевод",
        // ... остальные ключи ...
    }
};
```

3. Обновите выпадающий список в `index.html` (строка ~192):

```html
<option value="xx">🏳️ Language Name</option>
```

4. Убедитесь, что все ключи переводов совпадают с другими языками
